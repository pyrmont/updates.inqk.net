- I finally made [Listless](https://apps.inqk.net/listless) available in the App Store. I don't expect it to be used by anyone other than me but it's a free app so you don't really have much to lose if you do decide to check it out. If you're curious why I even wanted to build a to-do list app, I also wrote a [blog post](https://articles.inqk.net/2026/04/21/introducing-listless.html). It goes into a bit of detail about the rationale for Listless as well as my experience using LLM-based coding agents to make it.

- I mentioned [two weeks ago](https://updates.inqk.net/post/1775826960.html) that I had immediately started work on a second app. This one is a collaborative crossword app that I've been meaning to create for ages. I love doing crosswords but I find simple ones too easy and the more complex ones too frustrating. One thing I discovered during the pandemic is that if I work on a crossword together with another person, that drastically increases the ones I can complete. Since then  I’ve been wishing for some kind of app that would support that kind of solving. I had originally thought I might create a web app but I could not work out how I could do it without using a server (there are peer-to-peer technologies like [WebRTC](https://webrtc.org) but these still require a server to at the very least establish the communication). A native iOS app can get around these problems by using iCloud (or at least that’s my working assumption).
